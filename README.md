# Coursera Dump

## About script

This code gathers information about 20 different randomly selected courses from https://www.coursera.org/. 

## Requirements

Python 3.5

requests==2.11.1

openpyxl==2.3.5

beautifulsoup4==4.5.1

### Getting started

* Install modules
      
         pip install -r requirements.txt

* Run the script in a terminal

         python coursera.py
